# Ежедневник с задачами

Веб-приложение для управления проектами, задачами и заметками с красивым темным дизайном.

## Возможности

- ✅ Управление проектами (создание, удаление, переключение)
- ✅ Создание и управление задачами с отметкой выполнения
- ✅ Редактирование задач
- ✅ Заметки с автосохранением
- ✅ Темная тема с современным дизайном
- ✅ Все данные сохраняются в SQLite

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

Запустите приложение:
```bash
python app.py
```

Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## Структура проекта

- `app.py` - Flask приложение с API endpoints
- `database.py` - Модуль для работы с SQLite
- `templates/index.html` - HTML шаблон
- `static/css/style.css` - Стили в темной теме
- `static/js/app.js` - JavaScript для интерактивности
- `planner.db` - База данных SQLite (создается автоматически)

## Использование

1. **Проекты**: Создавайте проекты в левой панели, переключайтесь между ними
2. **Задачи**: Добавляйте задачи в левой части экрана, отмечайте выполненные, редактируйте двойным кликом
3. **Заметки**: Пишите заметки в правой части экрана - они сохраняются автоматически
