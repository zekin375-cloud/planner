# Быстрый старт: Компьютер как сервер, телефон как клиент

## Шаг 1: Установите зависимости

```bash
pip install -r requirements.txt
```

## Шаг 2: Запустите сервер на компьютере

**Windows:**
```cmd
start-server.bat
```

**Linux/Mac:**
```bash
chmod +x start-server.sh
./start-server.sh
```

Скрипт покажет IP адрес, например: `http://192.168.1.100:5000`

## Шаг 3: Убедитесь, что компьютер и телефон в одной Wi-Fi сети

## Шаг 4: Настройте приложение на телефоне

После установки APK:

1. Откройте приложение
2. При первом запуске введите IP адрес сервера (из шага 2)
3. Или измените `capacitor.config.json` перед сборкой APK:

```json
{
  "server": {
    "url": "http://192.168.1.100:5000"
  }
}
```

## Шаг 5: Готово!

Телефон будет отправлять все запросы на компьютер, данные хранятся на компьютере.

## Проверка

Откройте в браузере телефона: `http://ВАШ_IP:5000`
Должна открыться страница приложения.
