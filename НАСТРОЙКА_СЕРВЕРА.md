# Настройка: Компьютер как сервер

## Как это работает

- **Компьютер** = сервер (Flask) - хранит все данные
- **Телефон** = клиент - отправляет API запросы на компьютер
- Все данные в базе `planner.db` на компьютере

## Быстрая настройка

### 1. На компьютере:

```bash
# Установите зависимости
pip install -r requirements.txt

# Запустите сервер
start-server.bat    # Windows
# или
./start-server.sh   # Linux/Mac
```

Скрипт покажет IP адрес, например: `http://192.168.1.100:5000`

### 2. На телефоне:

**Вариант А: Через настройки в приложении**
- Откройте приложение
- Введите IP адрес сервера (из шага 1)

**Вариант Б: Перед сборкой APK**

Измените `capacitor.config.json`:
```json
{
  "server": {
    "url": "http://192.168.1.100:5000"
  }
}
```

Затем пересоберите APK.

### 3. Проверка:

Откройте в браузере телефона: `http://ВАШ_IP:5000`
Должна открыться страница приложения.

## Важно!

✅ Компьютер и телефон должны быть в **одной Wi-Fi сети**
✅ Файрвол должен разрешать подключения на порт 5000
✅ Сервер должен быть запущен на компьютере

## Решение проблем

**Телефон не подключается:**
1. Проверьте IP: `python get-local-ip.py`
2. Проверьте файрвол (разрешите Python)
3. Убедитесь, что в одной сети

**CORS ошибки:**
- Уже настроено в `app.py` (flask-cors)
